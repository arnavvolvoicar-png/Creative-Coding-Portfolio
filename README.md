# Creative-Coding-Portfolio
## Arnav Volvoicar 632491

This portfolio documents a series of creative coding experiments developed in p5.js, with additional use of Matter.js for physics‑based interaction. The brief required three themed experiments, each explored through multiple iterations to show technical progression, creative decision‑making, and an understanding of how code can be used to generate dynamic visual behaviour. Across the experiments, I moved from simple generative patterns to interactive systems and finally to a more complex physics‑driven simulation. Each experiment includes working code and commentary explaining the techniques used, the evolution of the idea, and the creative and technical challenges encountered along the way.
