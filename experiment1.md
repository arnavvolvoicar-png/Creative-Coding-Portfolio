# Experiment 1

In the base version of this experiment, I explored simple generative design using basic 2D shapes. The sketch produced a new composition every time the page refreshed, using randomness to control colour, size, and placement. This helped me understand how small changes in randomness can create large differences in visual outcomes. My focus at this stage was learning how p5.js handles shapes, layering, and colour, and how to use randomness in a controlled way to generate variety. Although this version was simple, it allowed me to experiment freely and build confidence with generative patterns.

[Random Shape Experiment](/1.1/index.html)

I moved away from pure randomness and introduced a grid‑based structure. This gave the design more order and rhythm while still allowing variation. I added mouse‑controlled spacing and size, which made the pattern interactive and allowed the user to influence the visual density and layout. This shift from randomness to structure helped me understand how user input can shape generative outcomes in real time. I also experimented with colour rules and spacing to create more intentional visual patterns.

[Control Spacing using mouse](/1.2/index.html)
[Animated Generative Pattern](/1.3/index.html)

In the final version, I shifted the project toward a more dynamic and game‑inspired visual system based on the CS2 case‑opening animation. I developed a scrolling rarity bar that spins and lands on a random item, turning the generative pattern into an interactive reveal mechanic. My first version worked but stopped awkwardly between items, so I improved it by adding snapping logic to ensure the bar lands perfectly in the centre of a square. I also added a masking system so the rarity colours are hidden during the spin, creating suspense and making the reveal feel more dramatic. AI helped me understand how to implement the snapping behaviour and refine the animation timing, but the concept, visual style, masking system, and overall behaviour were my own contributions. Through this version, I learned how animation, timing, and interaction can work together to create a polished, game‑like experience.
