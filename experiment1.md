# Experiment 1

In the base version of this experiment, I explored simple generative design using basic 2D shapes. The sketch produced a new composition every time the page refreshed, using randomness to control colour, size, and placement. This helped me understand how small changes in randomness can create large differences in visual outcomes. My focus at this stage was learning how p5.js handles shapes, layering, and colour, and how to use randomness in a controlled way to generate variety. Although this version was simple, it allowed me to experiment freely and build confidence with generative patterns.
